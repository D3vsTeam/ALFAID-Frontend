# ALFAID-Frontend



Git Clone
```
https://github.com/D3vsTeam/ALFAID-Frontend.git
```

Install Expo
```
npm install --global expo-cli
```

Install Packages
```
npm I
```
Run
```
  expo start 
  or
  npm start
```
Baixar aplicativo Expo e escanear QRCODE
